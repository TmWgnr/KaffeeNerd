- MongoDB
- React component testing
- Jest
- npm
- Git

### How to setup?

- Clone this repository
- Install all dependencies with `$ npm install`
- Run app in dev mode with `$ npm run dev`
- Server: [http://localhost:3000/](http://localhost:3000/)
